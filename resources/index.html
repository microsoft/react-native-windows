Repos

The following is a suite of offerings from Microsoft in the React Native space to help you target Desktop platforms as well as accelerate your development efficiency with React Native.

React Native for Windows

Github Repo

Official React Native for Windows framework for building native Windows applications with React.

React Native for macOS

Github Repo

Official React Native for macOS framework for building native macOS applications with React.

Fluent UI React Native (FURN)

Github Repo

FluentUI React Native is a JavaScript component library providing developers with controls that are in the Fluent Design System.

Dual Screen

Github Repo

Microsoft's offerings to streamline dual-screen cross-platform development using React Native. The modules in the repo are targeting Windows and Android.

React Native XAML

Github Repo

React Native XAML enables apps to use any XAML and WinUI native platform control.

VS Code RN extension

Github Repo

React Native extension for VS Code enables you to debug your code and quickly and run react-native commands from the command palette.

Hermes for Windows

Github Repo

Microsoft's fork of facebook/Hermes that brings Windows support to the lightweight JS engine for React Native.

App Center SDK for React Native

Github Repo

App Center allows you to get faster release cycles, higher-quality apps, and app insights to build what users want.

React Native Test App

Github Repo Sample

React Native Test App provides test apps for all platforms as a package.

React Native Gallery

Github Repo

React Native Gallery showcases the different controls available in React Native for Windows and community modules.
